# 分析模型选型矩阵（decompose 阶段参考）

> 迁移自 v1.0「问题拆解」。按"研究目的 × 研究对象维度"查出标配分析模型，用其 MECE 维度做子问题拆解骨架。

## 坐标编码
`坐标编码 = 「目的码-维度码」`
- 目的码：1=认知 2=监测 3=比较 4=判断 6=决策（5=验证 暂未启用）
- 维度码：1=产品能力 2=营销运营 3=客群体验 4=技术壁垒 5=商业模式 6=财务与资本运作 7=组织结构 8=供应链生态 9=市场基本面 10=产业链 11=竞争格局 12=宏观环境

## 认知（建立基础理解，穷尽刻画结构）
- 1-1 产品能力 → Kano 模型（基本/期望/兴奋 三层穷尽；可配 JTBD 任务拆解）
- 1-2 营销运营 → 4P/7P 营销组合（产品/价格/渠道/促销 +人/流程/有形展示；可叠加 STP）
- 1-3 客群体验 → 用户体验旅程地图（接触点 × 阶段：认知-购买-使用-留存；前台体验+后台支撑）
- 1-4 技术壁垒 → 经济护城河类型学（无形资产/转换成本/网络效应/成本优势/有效规模）
- 1-5 商业模式 → 商业模式画布 BMC（客户/价值主张/渠道/关系/收入/资源/活动/伙伴/成本 共9模块）
- 1-6 财务与资本运作 → 三张报表勾稽体系（经营/投资/筹资 × 资产-负债-权益）
- 1-7 组织结构 → 麦肯锡 7S（战略/结构/系统/共同价值观/技能/人员/风格）
- 1-8 供应链生态 → SCOR（计划-采购-生产-交付-退货 +赋能）
- 1-9 市场基本面 → TAM-SAM-SOM（总体可及/可服务/可获取 三层）
- 1-10 产业链 → 价值链/价值系统（上游-中游-下游 增值活动；可配微笑曲线）
- 1-11 竞争格局 → 战略群组图（两条关键战略维度把玩家穷尽聚类）
- 1-12 宏观环境 → PESTEL（政治/经济/社会/技术/环境/法律 六维）

## 监测（按时间序列追踪动态）
- 2-2 营销运营 → AARRR 海盗指标（获取-激活-留存-推荐-收入 漏斗）
- 2-3 客群体验 → 净推荐值 NPS（配 CSAT/CES 与留存曲线）
- 2-4 技术壁垒 → 技术 S 曲线（投入 vs 性能 时序；专利数量/引用为辅助）
- 2-5 商业模式 → 单位经济 LTV/CAC（按同期群随时间观察）
- 2-6 财务与资本运作 → 杜邦分析（ROE = 净利率 × 周转率 × 权益乘数）
- 2-7 组织结构 → 麦肯锡 OHI（9 类健康维度 + 人效/流失/管理跨度）
- 2-8 供应链生态 → SCOR 绩效指标
- 2-9 市场基本面 → Bass 扩散模型（创新+模仿 系数刻画 S 形渗透）
- 2-10 产业链 → 库存周期(基钦)（上中下游 价格/库存/产能利用率 时序）
- 2-11 竞争格局 → 份额时序 + HHI 集中度（判断趋于分散还是集中）
- 2-12 宏观环境 → 先行指标 + 美林时钟（PMI/利率曲线定位周期阶段）
- 注：2-1（监测×产品能力）无标配模型，借用一组产品质量/性能的时序指标。

## 比较（同口径对称比较多个主体）
- 3-1 产品能力 → 功能对标矩阵（行=能力维度，列=对比主体，强制对称打分；配 Kano 加权）
- 3-2 营销运营 → 声量份额 SOV（同口径 渠道/投放/转化 指标对比）
- 3-3 客群体验 → SERVQUAL 差距模型（RATER 五维：可靠/响应/保证/移情/有形）
- 3-4 技术壁垒 → 技术成熟度 TRL 对比（1–9 级；专利地图比卡位密度）
- 3-5 商业模式 → BMC 横向对比（同 9 模块并排定位差异点）
- 3-6 财务与资本运作 → 可比公司分析 Comps（EV/EBITDA、P/E 同口径对比）
- 3-7 组织结构 → 标杆分析法（管理跨度/层级/人效 同口径对标最佳实践）
- 3-8 供应链生态 → SCOR + ASCM 行业基准库（同口径 KPI 对标）
- 3-10 产业链 → 微笑曲线（沿产业链各环节比较附加值高低）
- 3-11 竞争格局 → 战略群组图 + 战略钟（价格 × 感知价值 对称比较定位）
- 3-12 宏观环境 → 波特钻石模型（生产要素/需求/相关产业/企业战略 四维；国别对比）
- 注：3-9（比较×市场基本面）无标配模型，借用同口径的市场规模/增速对比。

## 判断（定性评估、下结论）
- 4-1 产品能力 → RICE（触达×影响×信心÷投入 打分排序；配 Kano 判断哪类属性值得投）
- 4-3 客群体验 → SERVQUAL 差距模型（RATER 五维 期望-感知差距）
- 4-4 技术壁垒 → VRIO（有价值/稀缺/难模仿/有组织，判断能否带来持续竞争优势）
- 4-6 财务与资本运作 → 财务比率 + Altman Z（偿债/盈利/营运/成长 四类；Z 判断破产风险）
- 4-8 供应链生态 → 克拉杰里克矩阵（供应风险 × 利润影响，判断品类战略重要性）
- 4-9 市场基本面 → GE-McKinsey 吸引力维度（规模/增速/盈利性/壁垒/风险 加权评价）
- 4-10 产业链 → 利润池分析（各环节利润规模与利润率，判断价值与议价能力所在）
- 4-11 竞争格局 → 波特五力 + SWOT（现有竞争/新进入/替代/供方/买方 评价吸引力）
- 注：4-2/4-5/4-7/4-12（营销运营/商业模式/组织结构/宏观环境）无标配模型，借用对应"认知"或"比较"格的模型，并补一个判断性结论维度。

## 决策（输出可行动建议、做取舍）
- 6-1 产品能力 → RICE + BCG 矩阵（功能取舍排序 + 多产品组合资源分配）
- 6-3 客群体验 → 重要性-表现分析 IPA（重要性 × 表现 四象限定改进优先级）
- 6-5 商业模式 → 安索夫矩阵（现有/新 × 产品/市场：渗透/开发/拓展/多元化）
- 6-6 财务与资本运作 → NPV/IRR (DCF)（投资/并购选项排序；高不确定项配实物期权）
- 6-7 组织结构 → 组织设计 + RACI（职能/事业部/矩阵/网络 选型；RACI 落实权责）
- 6-8 供应链生态 → 克拉杰里克矩阵（采购策略：杠杆/瓶颈/关键/常规；自制 or 外购）
- 6-9 市场基本面 → GE-McKinsey 矩阵（市场吸引力 × 自身竞争力；选进入模式）
- 6-10 产业链 → 利润池 + 垂直整合（上/下游整合 还是聚焦高利润环节）
- 6-11 竞争格局 → 波特三战略 + 蓝海战略（红海三选一定位 or 蓝海 ERRC 开辟）
- 注：6-2/6-4/6-12（营销运营/技术壁垒/宏观环境）无标配模型，借用邻近格模型并补决策取舍维度。

## 灰格（无标配）规则
- 严格按坐标查表选型；命中灰格时，选最接近的邻近模型，并在 `model_index` 标注"无标配，借用 X-Y"。
- 用所选模型的维度做 MECE 骨架：尽量覆盖每一个维度，不重叠、不遗漏。
- 每个适用维度至少产出 1 个子问题；某维度对当前 focus 确实不适用时可省略，并在 `decomposition_logic` 说明原因。
